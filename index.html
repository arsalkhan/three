Three

Three is a SASS Framework for fast UX development. It's Modular CSS approach is way easier than any other current frameworks. It includes a lot of useful mixins that makes development a cheese cake. It's in the development process and any genius's contribution in development will be highly appreciated.

It consists of following folders: engine - controls all the includes of partial files core - contains all core styles of the web modules - files that all websites do not require vendor - vendor styles like fontawesome and bootstrap

engine

As the name describes, this directory contains all the files that are required to make this framework work. These files should not be edited unless you want to add some new feeatures e.g., mixins, in the framework. Only one partial file i.e., '_variables.scss' needs to be edited according to new project, e.g., colors.

core

All the project related styles are defined here. It contains many files, one for each major part in website, e.g., typography, buttons, forms, layout, etc.

modules

Modules are the files that are not required by all websites, e.g., every website does not need to be print friendly, so you can remove the print module.

vendor

This directory contains files by different vendors, the files that you do not need to edit, just include them and they will be added to your main css file, e.g., fontawesome.scss
