Three

Three is a SASS Framework for fast UX development. It's Modular CSS approach is way easier than any other current frameworks. It includes a lot of useful mixins that makes development a cheese cake. It's in the development process and any genius's contribution in development will be highly appreciated.

It consists of following folders:
engine - controls all the includes of partial files
core - contains all core styles of the web
modules - files that all websites do not require
vendor - vendor styles like fontawesome and bootstrap
